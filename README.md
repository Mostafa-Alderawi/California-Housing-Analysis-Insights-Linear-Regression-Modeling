# California Housing Price Analysis & Prediction

## Project Overview
This project explores the key factors influencing housing prices in California through exploratory data analysis (EDA),data visualization and feature engineering, culminating in a **Linear Regression model** to predict property values.

## Key Questions
- How does **median income** affect house value?
- How does **geographic location** (Latitude/Longitude) impact pricing?
- Do housing density features impact pricing?

## Tools & Libraries
-  Python
-  Pandas, NumPy
-  Matplotlib, Seaborn
-  Scikit-Learn

## Key Insights
- **Income is King:** Median income proved to be the strongest predictor of house prices.
- **Location Matters:** houses in island are the highest price , and houses in coastal areas are two times more than those in the inland.
- **Model Performance:** Our baseline Linear Regression model achieved an **R² Score of 0.65** and an **RMSE of $66,982**.

## Dataset
The dataset is sourced from the [Kaggle California Housing Dataset](https://www.kaggle.com/datasets/camnugent/california-housing-prices). It contains 20,640 records with 10 features.

## Project Structure
- `housing.csv`: Raw data.
- `california-housing-price-analysis-insights.ipynb`: Full analysis and modeling code.
